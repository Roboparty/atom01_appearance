🎨 设计理念

本仓库收录了 ATOM01 人形机器人原型机的全套外观件（外壳）设计资料。

在哈尔滨的研发历程中，我们的本体设计团队对市面上普遍存在的“工程优先”或“技术优先”的传统理念提出了大胆质疑。针对外观结构，我们确立了 “设计驱动工程” (Design-Driven Engineering) 的核心方针：

拒绝妥协的美学：外观不仅仅是覆盖件，而是机器人性格的延伸。我们追求线条的流畅与比例的和谐，力求在工业产品中注入艺术感。
严谨的结构支撑：所有美学造型均建立在严谨的技术评审与合理的内部结构基础之上。
高频协作的产物：这是设计团队与工程团队高频信息交流、无缝对接的成果，实现了从概念草图到工程落地的精准转化。

🤖 关于 ATOM01 外壳

ATOM01 是我们这一设计理念的首次实体化呈现。作为一台孕育时间短暂但意义重大的原型机，其外壳设计不仅满足了“能跑能跳”的功能需求，更是一次融合艺术与技术的工程实验。

虽然它已帮助我们积累了宝贵的制造与装配经验，但也暴露了大量关于材料选型、表面处理、模块化拆装及人机交互细节上的问题。这些“不完美”正是我们迭代下一代产品的核心驱动力。

🚧 已知问题与改进方向 (TODO)

作为原型机外壳，ATOM01 存在以下待改进点，欢迎社区贡献思路：

接缝控制：部分覆盖件之间的段差与缝隙均匀度需进一步优化。
走线问题：部分件存在走线问题，建议目前静态展示使用。
